# 文件夹城市（Folder City）

一个只在本地浏览器内运行的文件夹元数据可视化 MVP。选择文件夹后，应用将文件类型、大小和最后修改时间映射为可缩放、可平移、可筛选的二维 SVG 城市。

## 运行

```bash
npm install
npm run dev
```

开发服务器启动后，在浏览器中打开终端显示的本地地址。

## 验证与构建

```bash
npm test
npm run typecheck
npm run build
```

生产构建结果位于 `dist/`。

## 浏览器要求

选择文件夹依赖 File System Access API，因此需要支持该 API 的 Chromium 系浏览器（例如最新 Chrome 或 Edge）。不支持时，应用会说明该限制，且不会提供 ZIP 上传等替代入口。

## 隐私边界

- 只读取文件名、相对路径、大小、类型和最后修改时间等元数据。
- 不读取文件内容，不会调用文件内容读取 API。
- 不上传或保存任何文件内容或元数据；没有后端、账号、云同步或浏览器持久化。
- 数据仅保留在当前页面内存中，刷新页面即丢失。
